# 2026-07-10 — PCCodex 记忆目录

用户要求：本机 Codex 以后的记忆也要自动在 git 仓库上同步，并创建专用记忆文件夹，命名为 `PCCodex记忆`。

执行约定：
- `PCCodex记忆/` 是本机 Codex 专用长期记忆目录。
- 重要对话和项目决策优先写入该目录。
- `memory/MEMORY.md` 保留共享记忆总入口，并挂载本目录入口。
- `sync.ps1` 使用脚本所在目录作为同步仓库，避免写死旧电脑路径。
- 计划任务 `caiku-sync` 每 10 分钟自动执行 git pull/add/commit/push。

后续 Codex 启动重要任务时，应先同步 `caiku` 仓库，再读取：
- `memory/MEMORY.md`
- `PCCodex记忆/MEMORY.md`
