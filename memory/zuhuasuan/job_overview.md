## 业务模式

公司通过抖音小程序租赁手机给客户，大量客户租到手后断缴失联、变卖手机。我的工作不是跟客户打电话催收，是拿着公司出具的刑事控告书去派出所报案，以诈骗罪要求立案，通过公安机关施压迫使客户还款或还机。

## 核心打法（已跑通）

### 工作闭环
1. 带着控告书去派出所报案
2. 警察受理后不出具立案决定且未做任何调查的 → 第二天电话沟通，先给台阶（还钱还机就撤案），再要求联系被控告人，不同意就暗示信访
3. 信访/12389投诉 → 上面压下来 → 警察被迫打电话给被控告人
4. 被控告人收到警察电话 → 感受到司法压力 → 主动协商还款/还机

### 关键数据（五案经验）
- 跑了5个派出所，全部拿到报警回执
- 1个：警察联系上客户，客户主动还机（苹果13，价值约2000）
- 2个：直接出具不予立案通知书，未做任何调查
- 2个：待定

### 话术策略（给警察打电话时用）

**第一阶段（给台阶）：**
"警官，我们当初报案是联系不上对方才不得已来的，希望你们能帮忙查询对方别的联系方式，如果他能还钱或者还机子我们就不追究了。"

**第二阶段（上强度）：**
"按照正规流程，你们接到报案后要先核查，最起码联系被控告人询问情况。现在你们都没有联系对方就直接出具不予立案，这不符合程序规定。没办法我只能通过信访局或者12389来维护合法权益了。"

### 法律依据
- 《公安机关办理刑事案件程序规定》第169条：受理报案必须当场询问控告人并制作笔录
- 第174条：受理后应当进行审查，必要时进行调查核实（包括询问相关人员）
- 第178条：不予立案须经县级以上公安机关负责人批准，出具书面通知书
- 警察未经调查即出具不予立案 = 程序违规

### 策略核心
- 不是真要立案抓人，是让警察打电话施压
- 客户不怕催收公司，但怕警察
- 每一步都合法合规——报案是权利，复议是权利，投诉程序违规也是权利
